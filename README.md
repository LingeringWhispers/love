# 💖 爱心表白页面

一个浪漫的爱心表白网页，带有动画效果和背景音乐《告白气球》。

## ✨ 功能特点

- 🎨 粉色系浪漫配色
- 💓 3D动态爱心动画
- 💕 背景飘落爱心效果
- ✨ 点击页面生成爱心粒子
- 📝 模糊到清晰的表白文字动画
- ❤️ 全屏红色爱心跳动效果
- 🎵 背景音乐《告白气球》自动播放
- 📱 完美响应式适配

## ⚠️ 重要提示

### 添加《告白气球》音乐文件

由于版权原因，请自行准备周杰伦的《告白气球》音频文件：

1. 下载《告白气球》.mp3 文件
2. 将文件重命名为 `gaobaiqiqiu.mp3`
3. 将文件放到 `qianduan` 文件夹中（与 index.html 同级）

## 🚀 如何用手机访问

### 方法一：使用 Python 启动服务器（推荐）

1. **确保电脑和手机连接同一个 Wi-Fi**

2. **获取电脑的 IP 地址**：
   - Windows: 打开命令提示符，输入 `ipconfig`，找到"IPv4 地址"（类似 192.168.x.x）
   - Mac/Linux: 打开终端，输入 `ifconfig` 或 `ip addr`

3. **在当前目录启动服务器**：
   ```bash
   python -m http.server 8000
   ```

4. **在手机浏览器中访问**：
   ```
   http://你的电脑IP:8000
   ```
   例如：`http://192.168.1.100:8000`

### 方法二：使用 Node.js

1. **安装 http-server**：
   ```bash
   npm install -g http-server
   ```

2. **启动服务器**：
   ```bash
   http-server -p 8000
   ```

3. **手机访问**同方法一

### 方法三：使用 VS Code Live Server

1. 在 VS Code 中安装 "Live Server" 插件
2. 右键点击 `index.html`，选择 "Open with Live Server"
3. 确保电脑和手机在同一网络
4. 手机访问电脑 IP 地址和端口

## 🎵 关于音乐

### 默认音乐：告白气球
- 页面已配置为自动播放《告白气球》
- 请确保将 `gaobaiqiqiu.mp3` 文件放在同一目录下
- 由于现代浏览器限制，可能需要用户点击页面后才会开始播放

### 更换其他音乐
如果你想更换为其他音乐：

```html
<audio id="bgMusic" loop>
    <source src="你的音乐文件.mp3" type="audio/mpeg">
</audio>
```

将音乐文件放在同一目录下，然后修改 `src` 属性即可。

## 📝 自定义表白文案

在 `index.html` 中找到 `<div class="confession-text">`，修改里面的文字内容：

```html
<div class="confession-text" id="confessionText">
    <p>亲爱的，</p>
    <p>从遇见你的那一刻起，</p>
    <p>我的世界便有了不一样的色彩。</p>
    <!-- 在这里添加你的表白内容 -->
</div>
```

## 🎯 使用说明

1. 打开页面后，点击右下角"播放音乐"按钮开始播放背景音乐
2. 点击页面任意位置会生成爱心粒子特效
3. 点击"查看表白"按钮，表白文字会从模糊变清晰，同时弹出全屏爱心动画

祝表白成功！💕
